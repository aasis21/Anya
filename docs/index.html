v0 · build-out mode

GitHub Copilot, sitting next to your tabs.

Anya is an MV3 sidebar extension for any Chromium browser — Edge, Chrome, Brave, Vivaldi, Arc — talking to a local Node bridge that wraps @github/copilot-sdk. The same agentic Copilot you run in your terminal — streaming, tools, MCP — with a side helping of Playwright over your real, logged-in browser.

01 · What you get

The terminal Copilot, ported to your tabs.

  1. 01

    Real Copilot, real streaming.

    One CopilotClient and one CopilotSession per chat thread. Token-by-token streaming with proper tool-call lifecycle events.

  2. 02

    Multi-chat with persistence.

    Drawer of chats, each with its own bridge-side session. Pin, tag, search, rename, delete, export to Markdown. Survives restarts via chrome.storage.local.

  3. 03

    Browser context as input.

    The composer speaks three languages — / for client commands, @ for ambient browser context, # for named references.

  4. 04

    Playwright in your real browser.

    A drive_tab tool shells out to playwright-cli. Click, type, screenshot, extract — with the same auth your tabs already have.

  5. 05

    Inline tool cards.

    Every tool call renders as a VS Code-style card with args, progress, and a result preview. Click to expand.

  6. 06

    Live debug panel.

    A 🐛 button opens a trace of every Native Messaging frame and bridge log line. Click any row for the full payload.

02 · Composer language

Three prefixes.
Three cognitive moves.

Most chat UIs lump commands, context, and references into one mushy textbox. Anya splits them — because the user's intent is different in each case.

/

commands

"Do something to the sidebar."

Client-only. Never reaches the model. Keeps your prompt transcript clean.

  • /new · /clear · /rename
  • /pin · /tag add ux · /search
  • /export · /stop · /help
@

context

"Look at what I'm looking at."

Ambient browser state. Expanded inline before the bridge sees the prompt.

  • @tab · @selection
  • @tabs · @tab:id|query
  • @url · @title · @clipboard
#

references

"Look up the thing I labelled X."

Named, curated lookups — bookmarks, files, prior chats. Reserved namespace.

  • #bookmark/…
  • #chat/…
  • — planned —
03 · Install

One line. Any Chromium.

Requires PowerShell 7+, Node 20+, git, and a Chromium browser with developer-mode extensions.

pwsh 
# one-liner — clones to ~/Anya and sets everything up
> iwr -useb https://raw.githubusercontent.com/aasis21/Anya/main/install.ps1 | iex

› cloning aasis21/Anya → ~/Anya
› building bridge/  ✓
› building extension/  ✓
› bridge ping → PONG
› registered native-messaging host for:
   ✓ Microsoft Edge
   ✓ Google Chrome
   ✓ Brave

done. load ~/Anya/extension/dist in your browser.

Then, in each browser

  1. Open edge://extensions (or your browser's equivalent)
  2. Toggle Developer mode
  3. Load unpacked → pick extension/dist/
  4. Pin the action icon → click → sidebar opens
  5. Type ping → expect PONG

Switches

-InstallDir <path>where to clone (default ~/Anya)
-Branch <name>track a non-main branch
-Browsers edge,chromeregister a subset

Manual

> git clone https://github.com/aasis21/Anya
> cd Anya; ./setup.ps1   # or ./setup.sh

### Short description options
- Anya brings GitHub Copilot into your browser workflow to read tabs, organize context, and automate repetitive web tasks.
- Built on GitHub Copilot SDK, Anya helps you run browser-heavy workflows with your existing Copilot subscription.

### Maker comment (first comment)
I built Anya because half my work happens outside the IDE: PRs, dashboards, forms, tickets, docs, and too many tabs.

I wanted one place to run those workflows with the same Copilot subscription I already use, without buying another AI plan.

My favorite real workflows right now:
- Review a PR and draft comments from context across tabs
- Organize OneDrive files by project and year
- Fill performance evaluation forms using notes and examples
- Collect tax receipts from portals and summarize totals
- Turn manual browser steps into a reusable runbook

If you try it, reply with one repetitive browser workflow and I will turn it into a prompt pack.
